# PainRegister
The PCB of an electrostimulator for the study of the method of objective pain registration.

Technical specifications:
- input voltage supply 5V;
- booster scheme on the UC3843BD1013TR;
- USB-UART module;
- main MCU STM32F103C8T6TR.
